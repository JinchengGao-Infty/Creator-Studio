## Agent 使用方式

- 随时可以新开 agent：`hydra agent open <名字> --task <task_id>`
- 启动 codex：`hydra agent spawn <名字> --task <task_id>`
- 名字按任务起更清晰（如 `rust-project`、`ui-chapter`）
